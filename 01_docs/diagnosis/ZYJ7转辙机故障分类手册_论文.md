# ZYJ7 电液转辙机故障分类手册

> **来源**：董健和《面向智能运维的ZYJ7型转辙机故障诊断技术研究》(北京交通大学, 2023)
> **数据来源**：南宁轨道交通4号线楞塘村站微机监测系统实测三相电流数据（20ms采样周期）
> **编制日期**：2026-07-17

## 1. FMEA 方法论与故障筛选

论文采用 FMEA（Failure Mode and Effects Analysis，失效模式与影响分析）方法，对 ZYJ7 转辙机故障模式进行系统性识别与优先级排序。

### 1.1 RPN 风险优先数

故障优先级按 RPN（Risk Priority Number）排序：

```
RPN = 频度(O) × 严重度(S) × 检测等级(D)
```

| 维度 | 含义 | 取值范围 | 说明 |
|------|------|----------|------|
| **频度 O** | 危险事件发生的频率 | 1（不会发生）~ 10（经常发生） | 见表2-3 |
| **严重度 S** | 失效造成后果的严重程度 | 2.5（无关紧要）~ 10（灾难性） | 见表2-4 |
| **检测等级 D** | 探测失效原因/模式的能力 | 2.5（几乎可确定）~ 10（低） | 见表2-5 |

> **原则**：严重度最优先考虑，其次频度，最后检测等级。故障导向安全。

### 1.2 初始 11 种故障模式及 RPN 值

论文通过 FMEA 初步识别了 11 种故障模式，按 RPN 排序后选取前 8 种（F1-F8）作为诊断模型的目标故障类型。F9、F10、F11 因 RPN 值最低被排除。

| 标记 | 故障类型 | 频度 O | 严重度 S | 检测等级 D | **RPN** | 是否入选 |
|------|----------|--------|----------|------------|---------|----------|
| F2 | 道岔卡缺口 | 7 | 10 | 10 | **700** | ✅ 入选 |
| F5 | 卡阻空转 | 5 | 10 | 10 | **500** | ✅ 入选 |
| F4 | 油缸油压过低 | 4 | 7.5 | 10 | **300** | ✅ 入选 |
| F3 | 缓放阶段锁闭不顺畅 | 7 | 5 | 8.5 | **297.5** | ✅ 入选 |
| F1 | 启动解锁阶段异常 | 8.5 | 5 | 5 | **212.5** | ✅ 入选 |
| F6 | 室外二极管短路 | 3 | 10 | 4 | **120** | ✅ 入选 |
| F8 | SH6副机开闭器接点接触不良 | 2 | 6.5 | 8 | **104** | ✅ 入选 |

## 2. 8 种入选故障类型详解（含正常类别 F0）

论文最终对 1 种正常状态 + 8 种故障状态共 **9 种模式**进行分类。每种故障类型包含 60 条训练样本 + 30 条测试样本。

### F0 — 正常

| 属性 | 描述 |
|------|------|
| **故障类型** | 正常（无故障） |
| **产生原因** | — |
| **电流曲线特征** | 三相电流曲线符合正常动作形态（启动尖峰 → 解锁回落 → 转换平台 → 锁闭凹口 → 缓放"小台阶" → 归零） |
| **对应项目规则** | 无规则触发（所有规则通过） |
| **BP-Adaboost F1-Score** | 0.9791 |
| **易混淆故障** | — |

---

### F1 — 启动解锁阶段异常

| 属性 | 描述 |
|------|------|
| **故障类型** | 启动解锁阶段异常 |
| **产生原因** | 道岔基本轨和尖轨密贴过紧 |
| **失效模式** | 解锁部件卡阻 → 道岔解锁困难 |
| **电流曲线特征** | **启动解锁阶段功率和电流峰值明显偏大**，其余阶段正常 |
| **严重度** | 5（轻微）— 道岔动作可完成，易被忽视 |
| **检测难度** | 5（中等）— 通过电流曲线峰值可识别 |
| **对应项目规则** | **R4**（启动峰值偏高）、**R7**（解锁段偏高） |
| **BP-Adaboost F1-Score** | 0.9882 |
| **召回率** | 0.8811 |
| **易混淆故障** | — |
| **现场处理建议** | 检查尖轨与基本轨密贴状态，调整密贴值在标准范围内 |

---

### F2 — 道岔卡缺口

| 属性 | 描述 |
|------|------|
| **故障类型** | 道岔卡缺口 |
| **产生原因** | 转辙机检测杆的缺口值**过大或过小**，使转辙机无法锁闭 |
| **失效模式** | 道岔动作未完成 → 转辙机工作异常 |
| **电流曲线特征** | **初始电流很大，且没有形成缓放"小台阶"**（缓放段缺失或异常） |
| **严重度** | **10（灾难性）**— 转辙机无法锁闭，对行车安全有严重影响 |
| **检测难度** | 10（低）— 但曲线特征明显，实际可识别度高 |
| **RPN** | **700**（最高优先级） |
| **对应项目规则** | **R8**（缓放段异常/缺失） |
| **BP-Adaboost F1-Score** | 0.9671 |
| **召回率** | 0.8710 |
| **易混淆故障** | F3（缓放阶段锁闭不顺畅）— 两者都有缓放段异常特征 |
| **现场处理建议** | 检查转辙机检测杆缺口值，调整至标准范围（通常2±0.5mm） |

---

### F3 — 缓放阶段锁闭不顺畅

| 属性 | 描述 |
|------|------|
| **故障类型** | 缓放阶段锁闭不顺畅 |
| **产生原因** | 道岔基本轨和尖轨密贴过紧 |
| **失效模式** | 转换阶段克服较大阻力，导致转换过程不顺畅 |
| **电流曲线特征** | **转换阶段三相电流值约 1.9A，略高于正常值**（正常约1.6-1.7A），缓放阶段形态异常 |
| **严重度** | 5（轻微） |
| **检测难度** | 8.5（中等偏低）— 电流差异小，需与正常曲线对比 |
| **对应项目规则** | **R5**（转换段功率偏高）+ **R8**（缓放段异常） |
| **BP-Adaboost F1-Score** | 0.9437 |
| **召回率** | 0.8687 |
| **易混淆故障** | **F2**（道岔卡缺口）、**F6**（室外二极管短路）— 论文特别指出 F3 与 F6 的 AUC 面积十分接近，分类性能最差 |
| **现场处理建议** | 检查基本轨与尖轨密贴值，必要时进行调整；检查锁闭装置状态 |

---

### F4 — 油缸油压过低

| 属性 | 描述 |
|------|------|
| **故障类型** | 油缸油压过低 |
| **产生原因** | 油缸密封件松动，导致油压不足 |
| **失效模式** | 转换动力不足 → 转辙机工作异常 |
| **电流曲线特征** | **转换过程电流持续时间较短**（动作提前结束），电流幅值可能正常 |
| **严重度** | 7.5（严重） |
| **检测难度** | 10（低）— 仅从时长判断，实际检测困难 |
| **对应项目规则** | **R2**（动作夭折）、**R3**（动作时长偏差） |
| **BP-Adaboost F1-Score** | 0.9808 |
| **召回率** | 0.9801 |
| **易混淆故障** | F8（SH6副机接触不良）— 都有"动作时间短"特征 |
| **现场处理建议** | 检查油缸密封件，及时更换松动或老化的密封件；检查液压油位 |

---

### F5 — 卡阻空转

| 属性 | 描述 |
|------|------|
| **故障类型** | 卡阻空转（摩擦连接器打滑/空转） |
| **产生原因** | 转辙机机构卡阻（道岔有异物、尖轨卡阻、滑床板阻力大等） |
| **失效模式** | 电机空转 → 锁闭装置未锁闭 → 道岔动作未完成 |
| **电流曲线特征** | **转换阶段三相电流值约 1.6A**，但持续时间极长（打满录制窗口），电流持续不回落 |
| **严重度** | **10（灾难性）** |
| **检测难度** | 10（低）— 曲线特征明显 |
| **RPN** | **500**（第二优先级） |
| **对应项目规则** | **R1**（动作超时/打满窗口） |
| **BP-Adaboost F1-Score** | 0.9844 |
| **召回率** | 0.8541 |
| **易混淆故障** | F7（转换过程阻力过大）— 都有转换段电流持续特征，但 F5 时间更长且不回落 |
| **现场处理建议** | 检查道岔区域是否有异物；检查滑床板、尖轨与基本轨间是否有卡阻；检查摩擦连接器状态 |

---

### F6 — 室外二极管短路

| 属性 | 描述 |
|------|------|
| **故障类型** | 室外二极管短路 |
| **产生原因** | 转辙机表示电路发生异常，室外二极管短路 |
| **失效模式** | 表示电路异常 → 转辙机工作异常 |
| **电流曲线特征** | **转换阶段三相电流值较大，超过 2A**，明显高于正常值 |
| **严重度** | **10（灾难性）** |
| **检测难度** | 4（较高）— 电流幅值明显偏高，容易检测 |
| **对应项目规则** | **R5**（转换段功率偏高） |
| **BP-Adaboost F1-Score** | 0.9236 |
| **召回率** | 0.8522 |
| **易混淆故障** | **F3**（缓放阶段锁闭不顺畅）— 论文指出两者电流曲线特征相似程度大，模型分类性能最差 |
| **现场处理建议** | 检查室外二极管是否短路或击穿；检查表示电路各接点状态 |

---

### F7 — 转换过程阻力过大

| 属性 | 描述 |
|------|------|
| **故障类型** | 转换过程阻力过大 |
| **产生原因** | 动作杆移动不顺畅（滑床板缺油、杆件锈蚀、异物阻碍等） |
| **失效模式** | 转换阻力增大 → 转辙机工作异常 |
| **电流曲线特征** | **转辙机转换时间较长，转换阶段三相电流约 2A**（偏高） |
| **严重度** | 2.5（无关紧要）— 道岔可完成动作 |
| **检测难度** | 8.5（中等偏低） |
| **RPN** | 85（8 种中最低） |
| **对应项目规则** | **R3**（动作时长偏差）+ **R5**（转换段功率偏高） |
| **BP-Adaboost F1-Score** | 0.9588 |
| **召回率** | 0.9313 |
| **易混淆故障** | F5（卡阻空转）— 两者都有转换段电流偏高 + 时间延长特征 |
| **现场处理建议** | 检查滑床板润滑状态；检查动作杆是否锈蚀、变形；加注润滑油 |

---

### F8 — SH6 副机自动开闭器接点接触不良

| 属性 | 描述 |
|------|------|
| **故障类型** | SH6 副机自动开闭器接点接触不良 |
| **产生原因** | 动作过程电流或电压异常（SH6 副机转换锁闭器开闭器接点接触不良） |
| **失效模式** | 副机动作异常 → 转辙机工作不稳定 |
| **电流曲线特征** | **动作过程电流持续时间较短，且有一段较大值**（电流脉冲式波动） |
| **严重度** | 6.5（介于严重与轻微之间） |
| **检测难度** | 8（中等偏低） |
| **对应项目规则** | **R6**（转换段台阶突变）— 电流"一段较大值"对应功率台阶突变 |
| **BP-Adaboost F1-Score** | 0.9345 |
| **召回率** | 0.8423 |
| **易混淆故障** | F4（油缸油压过低）— 都有"动作时间短"特征 |
| **现场处理建议** | 检查 SH6 副机开闭器接点接触情况；清洁或更换接触不良的接点 |

## 3. 3 种排除的故障类型

论文中因 RPN 值过低或诊断优先级原因未纳入模型的 3 种故障：

### F9 — 同步电路故障

| 属性 | 描述 |
|------|------|
| **故障类型** | 同步电路故障 |
| **产生原因** | 同步电路断线 |
| **电流曲线特征** | 道岔转换到位前**某一相电流值变为 0A**，另外两相电流值升高 |
| **RPN** | 75 |
| **排除原因** | 频度极低（O=1），且三相缺相特征极其明显，不需要 ML 模型即可检测 |
| **对应项目规则** | 项目目前未覆盖（可用简单的"某相电流归零 + 另两相升高"检测） |

### F10 — 电源断电

| 属性 | 描述 |
|------|------|
| **故障类型** | 电源断电 |
| **产生原因** | 道岔控制电路中三相电源断电、空气开关跳闸、DBQ 故障 |
| **电流曲线特征** | 三相电流曲线**始终为 0A** |
| **RPN** | **20**（最低） |
| **排除原因** | 频度最低（O=1），且检测等级极高（D=2，几乎可确定），特征过于明显 |
| **对应项目规则** | **R0**（采集异常/全零曲线） |

### F11 — 油管漏油

| 属性 | 描述 |
|------|------|
| **故障类型** | 油管漏油 |
| **产生原因** | 密封元件损伤、螺丝松动 |
| **电流曲线特征** | TJ、BHJ 和电流持续时间短，无转换电流（动作未完成即中断） |
| **RPN** | 72 |
| **排除原因** | 频度极低（O=2），综合优先级低 |
| **对应项目规则** | **R2**（动作夭折） |
