## 1. 背景：当前算法的固定偏移量模型

项目当前的五阶段分割（`FeatureExtractor.cs` / `diag_reference_check.py`）完全基于 `spikeIndex` 和 `activeEnd` 的**固定偏移量**：

```
① 启动尖峰: [0, 15) 内最大值
② 解锁段:   [spikeIndex+2,  spikeIndex+14)    = 12 点 (0.48s)
③ 转换段:   [spikeIndex+20, activeEnd-40)
④ 锁闭段:   [activeEnd-40,  activeEnd-22)    = 18 点 (0.72s)
⑤ 缓放段:   [activeEnd-22,  activeEnd-2)     = 20 点 (0.80s)
```

**核心问题**: 解锁段固定 0.48s、锁闭段固定 0.72s，与物理实际严重不符。实际解锁阶段持续约 5s（3 台机），锁闭阶段约 2s。当前模型将大量解锁阶段的曲线错误地归入"转换段"，又将锁闭爬升归入"转换段"。

## 2. ZYJ7 转换过程的物理机制

### 2.1 设备配置

| 配置 | 道岔类型 | 尖轨 | 芯轨 |
|------|---------|------|------|
| **5机 (18号)** | 可动心轨辙叉 | 3台: 1×ZYJ7 + 2×SH6 | 2台: 1×ZYJ7 + 1×SH6 |
| **4机 (12号)** | 可动心轨辙叉 | 2台: 1×ZYJ7 + 1×SH6 | 2台: 1×ZYJ7 + 1×SH6 |

- **ZYJ7（主机）**: 含 AC 380V 三相电机 + 双向柱塞泵，通过闭式液压回路输出动力
- **SH6（转换锁闭器/副机）**: 无独立电机，通过油管共享主机液压动力，完成本牵引点的解锁→转换→锁闭

### 2.2 外锁闭装置的关键物理过程

每个牵引点有独立的外锁闭装置：锁闭杆、锁钩、锁闭铁、锁闭框、尖轨连接铁。

**解锁过程**（所有牵引点依次完成）：

```
电机启动 → 油缸走行 → 锁闭杆凸台开始移动
  → 凸台穿过空动距离（约 1.5-2.5s/点）
  → 凸台到达锁钩缺口位置
  → 锁钩失去支撑，落下（解锁完成）
  → J1 锁钩落 → J2 锁钩落 → J3 锁钩落（顺序解锁）
```

**关键**: 三个牵引点的锁钩**并非同时落下**，而是按 J1→J2→J3（或 J3→J2→J1）依次解锁。每次锁钩落下时，液压负载突变，在功率曲线上产生一个小尖波。因此**解锁阶段从电机启动一直持续到最后一把锁钩落下**，约 5s（3 台机）。

**转换过程**（所有锁钩落下后）：

```
所有锁钩已落下 → 尖轨自由平移（无锁闭阻力）
  → 从定位移动至反位（或反之）
  → 持续到尖轨接触基本轨（密贴开始）
```

**锁闭过程**（密贴后）：

```
尖轨接触基本轨（密贴拐点）
  → 锁闭杆凸台滑出锁钩缺口
  → 锁钩沿斜面爬升（增力机构：15.2mm 推程 / 7.6mm 动程，力放大 ~2×）
  → 锁钩被锁闭杆与锁闭铁夹持（锁闭完成）
  → 自动开闭器接点切换（油缸 17.4mm 断开原表示 → 锁闭行程 23.6mm 切断电机 + 接通新表示）
  → 1DQJ 缓放
```

## 3. 实测数据验证

### 3.1 夹锤曲线揭示锁闭段起点（密贴拐点）

夹锤试验（2mm/4mm 试验）：在尖轨与基本轨间插入铁板：
- **2mm 板**: 应能锁闭（安全）
- **4mm 板**: 不应锁闭（不锁闭即合格）

当插入 4mm 板时，转换阶段结束后尖轨无法密贴，电机持续空转，功率曲线在密贴拐点后**持续上扬**而非正常回落。

**4-J 夹锤曲线实测**：密贴拐点（功率斜率突变点）出现在 **≈9.8s**，而 4-J 基线总长 11.72s。

```
锁闭+缓放 = 11.72 - 9.8 ≈ 1.9s（含 ~0.8s 缓放，纯锁闭 ~1.1s）
```

> **注意**: 连续故障曲线（间隔 < 60s）只看第一个时间点。后续曲线因前次夹锤导致动程变短，拐点时间不具参考价值。

### 3.2 液压曲线文献 + 标准曲线验证解锁阶段

1-J 标准曲线（~3000 条正常曲线中位数融合）显示：
- 尖峰后存在多个小波动（对应 J1/J2/J3 解锁尖波）
- 功率在约 **5-6s** 后才进入稳定平台（所有锁钩已落下）
- 尾部锁闭爬升从约 **9.8-10s** 开始

## 4. 修正后的真实时间分布

### 4.1 五机配置（18号道岔，尖轨 3台）

```
0s      0.3s              5.2s                   9.8s      11.72s
│        │                 │                       │          │
├─启动──┤                 │                       │          │
│ 尖峰   ├─────────────────┼───────────────────────┼──────────┤
         │   解锁阶段       │     转换阶段           │ 锁闭+缓放 │
         │   ~4.9s         │     ~4.6s             │  ~1.9s   │
         │                 │                       │          │
         │ 油缸启动        │  尖轨自由平移          │ 密贴→锁死 │
         │ 外锁空动        │  (所有锁钩已落下)      │ →接点切换 │
         │ J1→J2→J3 锁钩  │  功率平稳 ~0.27kW     │ →1DQJ缓放 │
         │ 依次落下        │                       │          │
         │                 │                       │          │
         功率: 解锁尖波×3   功率: 平稳              功率: 爬升→回落
```

| 阶段 | 时长 | 占比 | 物理标志 |
|------|------|------|----------|
| 解锁阶段 | **~4.9s** | 42% | 电机启动 → 最后一把锁钩落下 |
| 转换阶段 | **~4.6s** | 39% | 所有锁钩落下 → 密贴拐点 |
| 锁闭+缓放 | **~1.9s** | 16% | 密贴拐点 → 接点切换 → 1DQJ 缓放 |
| **总计** | **~11.8s** | | |

### 4.2 四机配置 + 五机芯轨（均为 2台）

```
0s      0.3s              4.0s                   6.7s      8.7s
│        │                 │                       │          │
├─启动──┤                 │                       │          │
```

| 锁闭+缓放 | **~1.9s** | 22% |
| **总计** | **~8.7s** | |

### 4.3 完整对比

```
                5机尖轨(3台,18号)   5机芯轨(2台,18号)   4机尖轨/芯轨(2台,12号)
                ═══════════════     ══════════════      ════════════════════
解锁阶段            ~4.9s              ~3.7s               ~3.7s
转换阶段            ~4.6s              ~2.7s               ~2.7s
```

### 4.4 核心规律

| 因素 | 影响 | 时长差异 |
|------|------|----------|
| **锁钩数量** | 决定解锁阶段时长 | 3 把 ≈5s vs 2 把 ≈4s，差 ~1.2s |
| **动程长度** | 决定转换阶段时长 | 18号尖轨 ≈4.6s vs 其他 ≈2.7s，差 ~1.9s |
| **锁闭机构** | 基本恒定 | 不论配置均 ≈1.9-2.0s |

## 5. 当前算法与物理实际的差异

| 维度 | 当前固定偏移模型 | 物理实际 |
|------|-----------------|----------|
| 解锁段长度 | 0.48s (12点) | 3.7-4.9s |
| 解锁段含义 | 尖峰后的小平台 | 从启动到最后一把锁钩落下 |
| 转换段起点 | spikeIndex+20 (≈0.8s) | 最后锁钩落下后（≈4-5s） |
| 转换段终点 | activeEnd-40 (≈前1.6s) | 密贴拐点（≈前2s） |
| 锁闭段长度 | 0.72s (18点) | ≈1.1-2.0s |
| 分段依据 | 纯数学偏移 | 物理事件（锁钩落下/密贴接触/接点切换） |

### 具体后果

1. **解锁段均值 (refUnlockMean) 被低估**: 当前只统计尖峰后 0.48s 的功率，实际解锁阶段的功率波动（锁钩落下的尖波）被错误地算入转换段
2. **转换段均值 (refConvMean) 被污染**: 包含了大量解锁阶段的功率波动和锁闭爬升段的功率
3. **锁闭段均值 (refLockMean) 位置偏移**: 固定取 activeEnd-40→-22，不一定对准真实锁闭爬升区间
4. **R5/R6/R7/R8 规则的灵敏度受限**: 因为各阶段特征未能准确反映对应的物理过程

## 6. 算法改进方向

### 6.1 解锁段：多峰检测替代固定偏移

**目标**: 从启动尖峰后到最后一个锁钩落下尖波结束

**方法**:
- 在 `[spikeIndex+2, ~总长40%]` 区间内检测功率局部峰值（锁钩落下尖波）
- 3 台机应检测到 3 个尖波（J1/J2/J3），2 台机应检测到 2 个
- 最后一个尖波结束位置 = 解锁段终点
- 解锁段均值应包含所有尖波区间

### 6.2 锁闭段：密贴拐点检测替代固定偏移

**目标**: 密贴拐点 → 接点切换完成

**方法**:
- 在 `[activeEnd-80, activeEnd-20]` 区间内计算功率斜率（一阶差分）
- 斜率最大值位置 = 密贴拐点（功率开始持续爬升的时刻）
- 密贴拐点也是转换段的真实终点
- 锁闭段均值 = 密贴拐点 → 锁闭爬升峰值区间的均值

### 6.3 辅助：电流曲线接点切换检测

**目标**: 精确确定锁闭完成时刻

**方法**:
- 三相电流曲线尾部出现的几个小凸起 = 自动开闭器动接点打到位、电路切换
- 电流凸起位置可用于校准锁闭段终点
- 此方法需要电流数据，可作为可选增强

### 6.4 分段改造示意

```
改造前（固定偏移）:
  解锁 = [spike+2,  spike+14)     ← 0.48s 固定
  转换 = [spike+20, active-40)
  锁闭 = [active-40, active-22)   ← 0.72s 固定

改造后（物理边界）:
  解锁 = [spike+2,  lastHookDropEnd]            ← 多峰检测确定
  转换 = [lastHookDropEnd, contactInflection]   ← 密贴拐点确定
  锁闭 = [contactInflection, contactSwitchEnd]  ← 功率爬升+电流凸起确定
```

## 7. 与现有基线的兼容性

当前 `baselines.json` 中的特征值是"被污染的统计量"，而非真实物理量：

| 特征 | 当前统计值 (1-J 定→反) | 实际物理含义 |
|------|------------------------|-------------|
| `refUnlockMean: 0.306` | 尖峰后 0.48s 均值 | 仅 J1 解锁前段，遗漏 J2/J3 |
| `refConvMean: 0.305` | spike+20 → active-40 均值 | 混入 J2/J3 解锁波动 + 锁闭爬升 |
| `refLockMean: 0.302` | active-40 → active-22 均值 | 比真实锁闭段略早 |

**改进策略**: 新分段算法上线后，基线必须用新算法重新计算。新旧基线的特征值不可混用。
