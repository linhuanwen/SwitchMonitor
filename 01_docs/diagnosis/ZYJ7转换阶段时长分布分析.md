> **数据来源**：
> - 论文：董健和 (2023)，南宁轨道交通4号线，20ms采样周期，三相电流数据
> - 项目实测：3号水道岔 CSM2010 数据，25Hz (0.04s/点)，23,999 个动作事件，时间跨度 2025-12-13 ~ 2026-06-29
> - 项目算法：`FeatureExtractor.cs` 五阶段分割规格

## 1. 采样参数与分段模型对照

论文和项目使用不同的采样率和分段模型，需要先对齐：

| 参数 | 论文 | 项目 |
|------|------|------|
| **信号类型** | 三相电流 (A/B/C) | 三相功率 (kW) |
| **采样率** | 50 Hz (20ms/点) | 25 Hz (0.04s/点) |
| **分段模型** | **3 段**（启动解锁 → 转换 → 缓放） | **5 段**（尖峰 → 解锁 → 转换 → 锁闭 → 缓放） |
| **分段方法** | 电流幅值阈值：1.8A / 0.6A | 固定偏移量：基于 spikeIndex 和 activeEnd |
| **机器类型** | ZYJ7 单机 | ZYJ7 (主机) + SH6 (副机) 双机牵引 |

### 1.1 两套分段模型的映射关系

```
论文 3 段模型                    项目 5 段模型
══════════════                  ══════════════
┌──────────────┐                ┌──────────────┐
│ ① 启动解锁段  │ ←──────────→ │ ① 启动尖峰    │ ~0.25s
│              │               │ ② 解锁段      │ ~0.48s
│  (1.8A 阈值) │               │  (解锁段结束≈  │  0.73s total)
└──────┬───────┘                └──────────────┘
       │                        ┌──────────────┐
│ ② 转换段     │ ←──────────→ │ ③ 转换段      │ 主体
│              │               │  (spikeI+20   │
│  (1.8A→0.6A) │               │   → aE-40)    │
└──────┬───────┘                └──────────────┘
       │                        ┌──────────────┐
│ ③ 缓放段     │ ←──────────→ │ ④ 锁闭段      │ ~0.72s
│              │               │ ⑤ 缓放段      │ ~0.80s
│  (0.6A 阈值) │               │               │  1.52s total)
└──────────────┘                └──────────────┘
```

## 2. 项目实测：五阶段精确时长分解

> 采样率 25Hz，1 点 = 0.04s。以下以 `FeatureExtractor.cs` 规约为准计算。

### 2.1 阶段边界定义

```
时间轴 (以采样点序号表示，0 为第一个采样点)
│
├─ [0, spikeIndex]              → ① 启动尖峰区 (搜索前15点)
├─ [spikeIndex+2, spikeIndex+14) → ② 解锁段 (固定12点 = 0.48s)
├─ [spikeIndex+14, spikeIndex+20) →   过渡间隙 (6点 = 0.24s)
├─ [spikeIndex+20, activeEnd-40)  → ③ 转换段 (可变长度，主体)
├─ [activeEnd-40, activeEnd-22)   → ④ 锁闭段 (固定18点 = 0.72s)
├─ [activeEnd-22, activeEnd-2)    → ⑤ 缓放段 (固定20点 = 0.80s)
├─ [activeEnd-2, activeEnd]       →   尾部过渡 (2点 = 0.08s)
└─ [activeEnd+1, n-1]             →   尾零填充 (3~4点)
```

> **注**：首部还有 3-4 个零值点（录制窗口前余量）。

### 2.4 J型 vs X型 关键差异总结

| 对比维度 | 尖轨 J型 | 芯轨 X型 | 差异 |
|----------|----------|----------|------|
| 总动作时长 | **~11.7s** | **~8.6s** | J 比 X 长约 36% |
| 转换段点数 | **227 点** | **147 点** | J 多 80 点 (**3.2s**) |
| 转换段占比 | 77.2% | 68.5% | X 的锁闭+缓放占比更大 |
| 固定段总时长 | 2.24s (19%) | 2.24s (26%) | 完全相同（解锁+锁闭+缓放） |
| 启动尖峰 | 3.24 kW | 3.33 kW | 相近 |
| 转换段功率 | 0.301 kW | 0.259 kW | J 型转换段功率高 16% |
| 缓放段功率 | 0.213 kW | 0.207 kW | 相近 |

> **核心规律**：J/X 两类曲线的**固定段时长完全相同**（解锁 0.48s + 锁闭 0.72s + 缓放 0.80s = 2.00s），差异**全部来自转换段**。这是因为尖轨比芯轨更长更重，需要更长的转换时间。

## 3. 论文模型：三段式电流曲线时长分布（理论值）

> 论文使用 50Hz 采样（20ms/点），电流幅值阈值分段。以下为推断的对应关系（论文未给出精确秒数）。

### 3.1 三段式划分原理

```
电流曲线
  │
  ├─ 峰值后，电流持续 < 1.8A 的首次下降点 → T1 (启动解锁段结束)
  │     ┌─────────────────────────────────┐
  │     │  启动解锁段：~0.7s (约35点@50Hz) │
  │     └─────────────────────────────────┘
  │
  ├─ 从曲线末端向前搜索，电流连续 > 0.6A 的首次上升点 → T2 (缓放段开始)
  │     ┌───────────────────────────────────┐
  │     │  转换段：~8-10s (主体)              │
  │     └───────────────────────────────────┘
  │
  └─ T2 到曲线结束
        ┌───────────────────────────────────┐
        │  缓放段：~1.5s (约75点@50Hz)        │
        └───────────────────────────────────┘
```

### 3.2 与项目五段模型的数值对应

| 论文阶段 | 对应项目阶段 | 估算点数 (项目@25Hz) | 估算时长 | 占比 (J型) | 占比 (X型) |
|----------|-------------|---------------------|----------|-----------|-----------|
| **启动解锁段** | ①+② | ~18 点 | **~0.7s** | ~6% | ~8% |
| **转换段** | ③ | J:~227 / X:~147 点 | **J:~9.1s / X:~5.9s** | ~77% | ~68% |
| **缓放段** | ④+⑤ | ~38 点 | **~1.5s** | ~13% | ~18% |

### 3.3 论文中 8 种故障对时长的影响

> 根据论文表2-7 及电流曲线描述整理：

| 故障 | 启动解锁段 | 转换段 | 缓放段 | 总时长变化 |
|------|-----------|--------|--------|-----------|
| **F0 正常** | 正常 ~0.7s | 正常 | 正常 ~1.5s | — |
| **F1 解锁异常** | **峰值偏大**，时长可能正常 | 正常 | 正常 | ≈ 正常 |
| **F2 卡缺口** | **初始电流大** | 可能异常 | **缺失** ❌ | 缩短（无缓放） |
| **F3 锁闭不顺** | 正常 | 电流 ~1.9A (偏高) | **异常** ⚠️ | ≈ 正常或略长 |
| **F4 油压过低** | 正常 | **时长明显缩短** ⚠️ | 可能缺失 | **大幅缩短** |
| **F5 卡阻空转** | 正常 | **打满窗口不回落** 🔴 | 缺失 | **>31.6s**（打满790点） |
| **F6 二极管短路** | 正常 | **电流 >2A** 偏高 | 正常 | ≈ 正常 |
| **F7 阻力过大** | 正常 | **时间较长 + 电流~2A** | 正常 | **明显增长** |
| **F8 副机接触不良** | 正常 | **一段较大值** | 可能缩短 | **缩短** |

## 5. 诊断实用速查

### 5.1 按阶段定位故障

```
故障定位决策树（按发生阶段）:

启动解锁段 (0 ~ 0.8s)
├─ 峰值 > 基线×1.5? → F1 解锁异常 (R4) 或大功率启动
├─ 尖峰后无回落?   → 卡阻前兆，需结合转换段判断
└─ 峰值 = 0?       → F10 断电 (R0)

转换段 (0.8s ~ 动作结束前1.5s)
├─ 时长 > 31.6s?              → F5 卡阻空转 (R1) 🔴 最高优先级
├─ 时长 < 基线×60%?           → F4 油压过低 或 F2 卡缺口 (R2)
├─ 均值 > 基线×1.3?           → F6 二极管短路 或 F7 阻力过大 (R5)
├─ 台阶比 ≠ 1.0?              → F8 副机接触不良 (R6)
└─ 均值≈正常, 但逐点有脉冲?   → 隐性问题 (需 P1 逐点对比)

缓放段 (动作结束前 1.5s ~ 结束)
├─ TailMean = 0?              → F2 卡缺口 (R8) — 缓放段缺失
├─ TailMean 偏差 > 30%?       → F3 锁闭不顺畅 (R8)
└─ 正常                       → 正常

锁闭段 (动作结束前 1.5s ~ 0.8s)
└─ LockMean 异常?             → F3 锁闭不顺畅 (R9)
```

### 5.2 各故障对时长的典型影响量

| 故障 | 时长变化 | 典型值 | 检测灵敏度建议 |
|------|----------|--------|---------------|
| F5 卡阻空转 | **+170%** 以上 | >31.6s | 极高（无法错过） |
| F4 油压过低 | **-40%** 以上 | <7s (J型) | 高 |
| F7 阻力过大 | **+15~30%** | 14-15s (J型) | 中（需与正常波动区分） |
| F2 卡缺口 | **-30~50%** | 6-8s (J型) | 高（缓放段缺失是强特征） |
| F1 解锁异常 | ±5% 以内 | ≈正常 | 低（需看峰值而非时长） |
| F3 锁闭不顺 | ±5% 以内 | ≈正常 | 低（需看幅值而非时长） |
| F6 二极管短路 | ±5% 以内 | ≈正常 | 低（需看幅值） |
| F8 接触不良 | -10~20% | 9-10s (J型) | 中 |
