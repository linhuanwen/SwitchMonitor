# Slice 6: 全链路交互联动

> **状态（2026-07-08 代码审核）**: ⚠️ 大体实现。链路"转辙机→日期→时间→图表"完整可用。缺口：曲线加载在 UI 线程同步执行（无异步/防抖/取消，快速连点与大 JSON 两条验收未落实）；状态栏缺动作数与扫描时间；无上一动作时下排图表留空白、无提示文字。

## Type

AFK

## Blocked by

## What to build

串联用户操作的完整交互流程：转辙机选择 → 日期加载 → 时间列表 → 自动选最近时间 → 图表渲染。实现顶部工具栏复选框控制系列显隐，以及所有边界状态（无数据、加载中、选中空）的处理。

### 完整交互流程

```
启动程序
  → 加载 config.json → 侧边栏显示转辙机列表
  → 用户点击"1-1"
      → C# OnSwitchSelected("1-1")
      → 读取 parsed_data/1-1/index.json → 获取可用日期列表
      → 侧边栏刷新日期选择器，默认选最近日期
      → 读取 parsed_data/1-1/YYYY-MM-DD.json
      → 侧边栏刷新时间列表（降序），首项自动选中
      → 取 首项 + 第2项 作为 [当前时间, 上一时间]
      → 构建 chartDataJson → InvokeScript("loadChartData", ...)
  → 用户点击时间列表中另一项
      → C# OnTimeSelected(ts)
      → 取 选中项 + 选中项下一项(更早) 作为 [当前, 上一]
      → 重新构建 chartDataJson → 更新图表
  → 用户点击侧边栏其他转辙机 → 重复以上流程
  → 用户切换日期 → 时间列表更新 → 同上
```

### 工具栏复选框

顶部工具栏通过另一个 WebBrowser 或直接在 `charts.html` 上方加 JS 绘制的 checkbox 行：

```
□A相电流  □B相电流  □C相电流  □功率  |  □报警上限  □保持量程
```

- 复选框默认全选
- 点击复选框 → JS 调用 `series.hide()` / `series.show()`
- "报警上限" → 切换 `yAxis.plotLines` 显隐
- "保持量程" → 切换 `yAxis.min/max` 固定/自适应

### 边界状态处理

| 状态 | 侧边栏 | 图表区 |
|------|--------|--------|
| 未选转辙机 | 时间列表为空，显示"请选择转辙机" | 显示"请选择转辙机查看曲线" |
| 选中转辙机无该日数据 | 时间列表为空，显示"该日无动作记录" | 保持上一条曲线或空状态 |
| 日期列表为空 | 显示"暂无数据" | 同上 |
| 当前时间无上一时间 | 上排两个图表显示"无上一动作" | 仅下排有数据 |
| 数据加载中 | 转辙机列表灰显 | 显示加载指示器 |
| 解析出错 | 错误信息显示在状态栏 | 图表显示错误提示 |

## Further notes

- 转辙机点击后如果在 500ms 内再次点击，取消上次加载（debounce）
- 数据加载用 `BackgroundWorker` 或 `Task`，完成后 `Control.Invoke` 回 UI 线程
- 状态栏用 `StatusStrip` + `ToolStripStatusLabel`
