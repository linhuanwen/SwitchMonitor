本地接收目录数据分析说明
========================

文件夹：shuju/本地接收目录扳动
-------------------------------
文件格式：Digit(YYYY_MM_DD_HH).dat
数据类型：数字量事件（道岔/继电器/表示点状态变化）
已转换至：shuju/本地接收目录扳动_parsed/*.csv

CSV列：
  timestamp    Unix时间戳
  datetime     可读时间
  record_type  记录中包含的点数（1/2/3/4/5/6/7/8/10/11/20）
  raw_value    原始16位值
  state_byte   高字节（状态/质量码）
  point_id     低字节（点号）

统计：
  文件数：84
  总事件数：13,667
  时间范围：2026-06-26 00:00:00 至 2026-06-29 11:01:00
  最常见状态码：0x00（7645次）、0x2f（4676次）
  最活跃点号：168、170、174、175、184、185、186、187（各1300+次）


文件夹：shuju/本地接收目录表示
-------------------------------
文件格式：DCBSDYAnalog(YYYY_MM_DD_HH).dat
数据类型：模拟量测量（疑似 DC 表示电压 / 电源电压）
已转换至：shuju/本地接收目录表示_parsed/DCBSDYAnalog*.csv

CSV列：
  timestamp         Unix时间戳
  datetime          可读时间
  channel           通道号（通常为0）
  measurement_type  测量类型/测点编号（1-19, 32）
  value             浮点数值（单位疑似伏特 V）

统计：
  文件数：84
  总样本数：492,672
  时间范围：2026-06-26 00:00:00 至 2026-06-29 11:54:53
  数值范围：0.00 - 227.68
  主要测量类型：
    type 1: 170,490 样本，均值 44.34 V
    type 2: 147,503 样本，均值 48.43 V
    type 3:  91,492 样本，均值 47.99 V


备注
----
1. 两个文件夹内均发现了 Digit(*).dat 文件，其中“本地接收目录表示”文件夹中同时存在 DCBSDYAnalog 和 Digit 两类文件。
2. 数字量点号 168-187 等高频点可能是道岔表示继电器或相关采集点；具体含义需对照监测软件点表。
3. 模拟量最大约 228 V，可能是直流电源母线电压、继电器线圈电压或整流后电压。
